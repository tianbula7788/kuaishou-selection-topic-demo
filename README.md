# 快手选品瓷片区与商品专题 Demo

本仓库包含快手选品“运营瓷片区与商品专题”需求文档、原型绘制指令稿和可交互静态 demo。

## 在线预览

GitHub Pages 启用后，可通过以下路径预览：

```text
https://<github-user>.github.io/<repo-name>/demo/
```

## 本地预览

```bash
cd demo
python3 -m http.server 4173
```

打开：

```text
http://127.0.0.1:4173/
```

## 目录

- `demo/`：可交互静态 demo
- `快手选品瓷片区与商品专题需求说明.md`：PRD 文档
- `墨刀原型绘制指令稿-快手选品瓷片区与商品专题.md`：原型绘制指令稿
